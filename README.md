# My Profile App

Aplikasi profil interaktif lintas platform (Android, iOS, dan Desktop) yang dibangun menggunakan Compose Multiplatform.

--- 

## Features

- Profile section with photo
- Contact information
- Skills section
- Send Message button
- Add Friend button
- Animated button appearance
- Dark aesthetic UI

---

## Technologies

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
